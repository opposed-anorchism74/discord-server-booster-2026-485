## 🛠️ Configuration and Usage

The application requires a few steps to connect to your Discord server.

### Initial Setup
When you open the app, look for the settings menu. You need your Discord Bot Token to link the app to your server. Copy your token from the Discord Developer Portal. Paste this key into the settings field in the app. Save your changes.

### Managing Server Boosts
The software automates the boosting process. Navigate to the Boost Manager tab. Choose the target server from the list. Select the number of boosts you need. Click Start. The app handles the rest. It syncs tokens and updates your server stats automatically.

### Automated Rewards
Reward your top members with the Reward Manager. Set rules for your server, such as boosting milestones or invite counts. The app tracks member activity. It assigns roles when a member hits a goal. This feature keeps your community engaged and active.

### Invite Tracking
The Invite Tracker keeps a log of all incoming members. It identifies which member sent the invite. You can see your Top Inviters list in the dashboard. Use this data to run contests or promotions.
